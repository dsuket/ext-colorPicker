Color Picker Form for Sencha Touch
===================================

This is color picker component for Sencha Touch 1.1.

Component
---------
- colorForm.js<br>
Color picker component. This has color pick button and color filed.

- colorSet.js<br>
Set of colorForm. This can have some colorForm, and sort forms.


Required
--------
- Sencha Touch 1.1<br>
you can download at <a href="http://extjs.co.jp/products/touch/download.php">here</a>


not support for Sencha Touch 2.x.<br> 
I hope someone do :)
